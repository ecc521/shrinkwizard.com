Transparent OS Compression

Utilizes built-in Windows (LZX, XPRESS) and macOS (LZFSE, ZLIB, LZVN) filesystem compression. Files remain fully accessible to all apps without needing to be extracted.

Pixel-Perfect Image Optimization

Optimizes JPEGs and PNGs by reorganizing their internal data structure without altering a single pixel or changing the file format. All metadata is preserved by default.
