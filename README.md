# Shrink Wizard Website

This repository contains the marketing site and landing page for **[Shrink Wizard](https://shrinkwizard.com)**.

## About Shrink Wizard

Shrink Wizard is a highly-optimized desktop application designed to permanently reclaim disk space through **100% lossless** file compression. Our goal is to make advanced data compression accessible through a dead-simple, beautiful user interface.

### Core Features

- **Zero Data Degradation:** Never compromises on visual or audio fidelity. All EXIF data, ICC color profiles, and file metadata are perfectly preserved bit-for-bit.
- **Smart Scanner:** Proactively scans your system to discover compressible folders and accurately estimate disk savings before you begin.
- **Advanced Encoders:** Automatically utilizes state-of-the-art native tools (such as Archival JXL transcoding and Transparent System Compression) for maximum efficiency.
- **Seamless UX:** Absolutely no command-line configuration required. It securely bundles all required tools and binaries for a 'one-click' experience on your OS.
